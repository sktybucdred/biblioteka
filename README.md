# Biblioteka - wersja podstawowa

Konsolowa aplikacja biblioteki napisana w Pythonie z wykorzystaniem
programowania strukturalnego (bez klas, bez zewnetrznych bibliotek).

## Funkcje

- Logowanie uzytkownika (max 3 proby)
- Przegladanie katalogu ksiazek
- Wypozyczanie ksiazek
- Podglad wlasnych wypozyczen

## Uruchomienie

```bash
py main.py
```
